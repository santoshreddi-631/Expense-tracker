# Expense Tracker

A simple Expense Tracker web application built using **HTML, CSS, and JavaScript**.

## Features
- Add income and expenses
- Track total balance
- View transaction history
- Simple and responsive UI

## Technologies Used
- HTML
- CSS
- JavaScript

## Important Note
The project contains image assets used inside the application.

If the images are not displaying correctly after cloning or downloading the project, please check the image source paths in your HTML/CSS/JS files because the folder structure or image paths may be different on your system.
